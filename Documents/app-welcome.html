Main Window

The appearance of the application on start-up: the main viewing area displays the "welcome" screen when there is no current project in progress. The window is divided into five parts (from top to bottom) main menu bar, convenience tool bar, the image viewer pane, the command button bar, and the status bar. The various menu items, and their corresponding tool bar buttons gray out if not actually required. as do the command buttons. The application window is resizable via the drag corner in the lower right-hand corner.

Both the main menu and toolbar are dockable, both within the main window, or outside of it.

 

 

Main Menu

File Menu

New Project:

This button and its tool bar button (light bulb icon) starts a new project. It may be clicked multiple times to do different designs without having to restart the application every time.

Save as...:

Invokes the file save dialog when a project is completed. Corresponding tool bar button: diskette icon.

Print...:
Invoke the print dialog to make hard copy of the project. There is also a tool bar button for this: printer icon.

Cancel:

This button resets the entire application, putting it into its initial start-up configuration. Use in case of a serious, but non-fatal run time error, or to start new projects. No corresponding tool bar or command buttons.

Exit:

Exit the application. Corresponding command button: Exit. Will prompt if there is a complete project which hasn't been saved yet

Project Menu

Passive BPF:

This menu item selects a passive bandpass filter. This being the Butterworth derived passive filters.

Active BPF:

Selects an active BPF design, either multi-feedback or state variable.

Passive Fltr:

Selects a passive filter design.

Active Fltr:

Selects an active filter design based on the Sallen-Key topology.

Help Menu

Contents...:

Display the application on-line help. This invokes a web browser to display this help.

About...:

Show the "About" dialog box.

Dolphins...

:
A special message from the developers.

Tool Bar

The tool bar buttons duplicate some of the more common main menu items. The buttons are icon only. However, hovering over a button will bring up a brief tool tip, and a more detailed description appears in the status bar.

Viewer

The welcome screen and completed projects are displayed in this pane. Project files are saved as *.gif images. These may be saved to the hard drive and/or printed as hard copy. These files may be viewed with almost any image viewer program, so that it is not necessary to start up the application if you need to consult the results of a completed project.

Command Buttons

OK:

This button finalizes a project. Will prompt if there are any entry errors. Project results are shown on the main window.

Clear:

Removes all user entered data without resetting the whole project.

Exit:

Exit the application. Will prompt if there is any unsaved data.

Status Bar

The status bar has several text display areas. The main one shows the application status as well as displaying more verbose tool tips. Project status is also shown here. There is a convenient LED icon.
LED gray: no project started yet.
LED off: Project in progress or complete, but not saved.
LED on: Project complete and saved.
